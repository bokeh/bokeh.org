Logo usage

When using the logo, please use the following directives:

  • DO make the logo large enough: at least 1 inch (2.54 cm) for print at least and 72 px for web
  • DO leave enough margin around the logo - at least around triple the width of the bar in the ”k” character;
  • Do NOT apply transparency to the logotype;
  • Do NOT change the logo proportion by distorting it;
  • Do NOT place text or other elements over the logo.

Citation

If you find Bokeh useful in your work, a citation is always appreciated.

To cite Bokeh in publications, use:

Bokeh Development Team ({{ site.time | date: '%Y' }}). Bokeh: Python library for interactive visualization
URL https://bokeh.org


A BibTeX entry for LaTeX users is:

@Manual{,
title = {Bokeh: Python library for interactive visualization},{% raw %}
author = {{Bokeh Development Team}},
year = {{% endraw %}{{ site.time | date: '%Y' }}{% raw %}}{% endraw %},
url = {https://bokeh.org/},
}
